# my :heart: dotfiles

My personal dotfiles build from scratch!

## Requirements

Install zsh:
```zsh
brew install zsh
chsh -s /bin/zsh
```

Install rcm:
```zsh
brew tap thoughtbot/formulae
brew install rcm
```

To use the provided [```Brewfile```](/homebrew/Brewfile) install ```brew bundle```:
```zsh
brew tap Homebrew/bundle
brew bundle
```
